# Event Sphere: A Unified Academic Event Aggregation Portal

## Project Description
Event Sphere is a comprehensive web-based platform designed to aggregate, manage, and discover academic events seamlessly. It serves as a centralized hub where students, faculty, and administrators can easily find and organize hackathons, seminars, workshops, and other academic activities. By bridging the communication gap within educational institutions, Event Sphere ensures maximum participation and streamlined event management.

## Project Structure
- `frontend/`: Contains the Next.js application for the user interface, utilizing Tailwind CSS for styling and Framer Motion for animations.
- `backend/`: Contains the Node.js and Express backend, connected to a MongoDB database to handle API requests and data storage.
- `Literature/`: Contains research materials, articles, and references used during the development of this project.

## Technologies Used
- **Frontend**: Next.js (React), Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose)
- **Authentication**: JWT (JSON Web Tokens) & bcryptjs
