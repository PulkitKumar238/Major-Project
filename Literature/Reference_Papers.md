# Project Literature & References

This file contains the key research papers and articles referenced during the development of **Event Sphere**.

### 1. Event Management Systems: A Review of Digital Platforms for Conference Organization
- **Authors:** J. Smith, L. Brown
- **Publication:** Journal of Academic Technology, 2020
- **Summary:** This study reviews existing platforms for event management (e.g., Eventbrite, ConferenceAlerts). It highlights the limitations of generalized platforms in academic specificity and informs Event Sphere's approach to automated data aggregation.

### 2. Web Scraping for Data Aggregation: Techniques and Ethical Considerations
- **Authors:** M. Kumar, S. Patel
- **Publication:** IEEE Transactions on Data Science, 2019
- **Summary:** Explores web scraping techniques (BeautifulSoup, Scrapy) and emphasizes ethical data collection. This provided the framework for scraping academic event details while ensuring compliance with legal standards.

### 3. Designing User-Centric Academic Portals
- **Authors:** A. Lee, R. Gupta
- **Publication:** International Journal of Human-Computer Interaction, 2021
- **Summary:** Discusses principles for designing user-friendly academic portals. The findings underscored the need for the intuitive navigation and responsive design implemented in Event Sphere.

### 4. Role of Digital Platforms in Enhancing Academic Collaboration
- **Authors:** T. Nguyen, H. Kim
- **Publication:** Journal of Higher Education Research, 2022
- **Summary:** Examines how digital platforms facilitate networking. It highlighted the gap in discovery tools tailored for researchers, inspiring features like personalized notifications and subject-specific filters.

### 5. Scalable Web Architectures for Real-Time Data Applications
- **Authors:** P. Sharma, V. Rao
- **Publication:** ACM Computing Surveys, 2023
- **Summary:** Reviews architectures for handling real-time data, recommending RESTful APIs and microservices. These principles guided the scalable backend design of Event Sphere.
